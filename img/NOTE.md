    妈的，我倒要听听这是个什么鬼东西 :(

### cls-0 正常状态

```
[听感]
- 关节骨骼来回磨合，或者说磨牙齿的声音 (疑似drive端数据，疑似有两种采样率)
- 空气或风扇声音，无明确声源 (疑似fan端数据)

[时域]
- 动态大、毛刺多，大小脉冲周期较为规律 (drive端)
- 纯空气声，无回声、无目标 (fan端)

[频域]
- 800~1.2kHz 断续咕噜声
- **480Hz** 暗线
```

- 局部无规律周期，也可能是样本白噪声太大

[频域]
- 1.1~1.5kHz 断续咕噜声 (空气化的)

### cls-2 外圈故障

```
[听感]
关节骨骼来回磨合，或者说磨牙齿的声音，频率相对内圈故障更慢

[时域]
- 整体呈现为平整的毛毛虫
- 局部易见规律元音状周期
- 有时可见毛刺，为不规律连续脉冲

[频域]
- 1.4kHz/1.38kHz 暗线
- 800~1.1kHz 断续咕噜声 (空气化的)
  - 930Hz
  - 860Hz
- 640, **480**, 360, 220Hz 暗线
- 其余为空气白噪声
```

### cls-3 滚动体故障

```
[听感]
似大型虫鸣、高频机械振动噪声

[时域]
- 整体呈现为平整的毛毛虫
- 局部可见中前元音 e 状小周期，但不太规律

[频域]
- 2.8kHz 蜂鸣
- 1.36~1.46kHz 蜂鸣带 (四条暗线)
  - 1.46kHz
  - 1.42kHz
  - 1.38kHz
  - 1.34kHz
- **480Hz** 蜂鸣
- 330Hz 蜂鸣
- 100~240Hz 蜂鸣带 (空气回声，很多暗线)
  - 220Hz
  - 180Hz
  - 120Hz
- 其余为空气白噪声
```
